Reboot CSS

La première partie du CSS cde bootstrap est un "reboot", c'est un dire toute une partie CSS qui permet de remettre en mode web et accessible les éléments normalement couvert par le CSS du navigateur, qui n'est pas accessible par défaut.

La police de caratère est modifiée, passant d'une police à empatement non adaptée à l'affichage à l'écran par une police sans empatement. La taille de la police par défaut est augmentée ainsi que l'interligne.

Tous les éléments sont par défaut mis en calcul de taille avec les bordures, alors que le CSS navigateur gère les tailles des éléments par leur contenu. De plus, la marge extérieure de body est supprimée.

Premièrement, il gère les titres, puis les paragraphes sont modifiés.

Les liens sont modifiés.

Le séparateur horizontal est <hr /> modifié

Il est possible de changer la couleur du séparateur en appelant des classe boostrap de bordure

Les listes ordonnées et non-ordonnées sont aussi modifiées

  • La marge extérieure haute est supprimée
  • Les liste comprises à l'intérieur d'élément de liste n'ont pas de marge extérieure basse

Il est possible de changer les couleurs, l'apparence, et d'ajouter des bordures aux tableaux, en utilisant d'autres classes bootstrap.

Au niveau des couleurs, on utilise encore les mots clef de couleur bootstrap :

  • primary : bleu
  • secondary : gris
  • success : vert
  • warning : orange
  • danger : rouge
  • dark : noir

Il est possible d'agir soit sur le tableau en entier, soit sur une ligne, voir une cellule.

Différentes classes sur les différentes parties du tableau
La ligne est en warning Comme la tr est en warning, la th et la td héritent du style css Comme cette td a la classe "table-primary", le style bleu des tableaux surcharge le style normalement hérité de la tr Cette td hérite du style parent, pas de la td précédente (élément frère / soeur ou "sibling")

Les formulaires

Les formulaires comportent toute une liste de classes qui permettent de mettre facilement en forme les champs de saisie.
De plus, Bootstrap permet de customiser les contrôles de formulaires, et cela automatiquement grâce au JS, et à l'utilisation des classes adéquates.
